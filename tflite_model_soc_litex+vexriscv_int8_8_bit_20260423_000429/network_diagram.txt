## Fluxo de Dados
1. Entrada escalar (1 valor)
2. Expansão para 16 neurônios (Dense + ReLU)
3. Processamento intermediário (Dense 16x16 + ReLU)
4. Redução para saída escalar (Dense 16x1)
5. Saída quantizada em 8-bit

Esta é uma rede neural simples do tipo MLP (Multi-Layer Perceptron) com 3 camadas densas e ativações ReLU, projetada para processamento de dados escalares.
